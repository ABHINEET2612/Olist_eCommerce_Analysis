# Dataset Source

This project uses the Brazilian E-Commerce dataset provided by Olist.

## Source
- Platform: Kaggle
- Dataset: Brazilian E-Commerce Public Dataset by Olist
- Link: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce/data

## Description
This dataset contains over 100,000 orders from 2016 to 2018 across multiple marketplaces in Brazil.

It includes:
- Orders and order status
- Payments and payment types
- Customer details
- Product and category information
- Reviews and ratings
- Delivery and shipping data

## Notes
- The dataset consists of multiple CSV files
- It is publicly available on Kaggle
- Due to size and best practices, raw files are not stored in this repository
